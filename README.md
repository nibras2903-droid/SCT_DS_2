## 📌 Task 02: Data Cleaning & Exploratory Data Analysis (EDA)

### 📖 Objective
Perform **data cleaning** and **exploratory data analysis (EDA)** on the Titanic dataset to understand the data, identify missing values, discover relationships between variables, and uncover meaningful patterns that influenced passenger survival.

---

## 📂 Dataset

**Dataset Used:** Titanic Dataset

- Source: Kaggle
- File: `Titanic-Dataset.csv`

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📋 Steps Performed

### 1. Data Loading
- Imported the Titanic dataset.
- Displayed the first few rows.
- Checked dataset dimensions.

### 2. Data Cleaning
- Identified missing values.
- Filled missing values in numerical columns.
- Filled missing values in categorical columns.
- Removed duplicate records.
- Converted data types where necessary.

### 3. Exploratory Data Analysis (EDA)
Performed analysis on:

- Survival distribution
- Gender vs Survival
- Passenger Class vs Survival
- Age Distribution
- Fare Distribution
- Embarked Port Distribution
- Correlation Heatmap
- Pairplot of numerical features
- Survival based on Age and Passenger Class
- Family Size analysis

---

## 📊 Visualizations

The project includes:

- Bar Charts
- Count Plots
- Histograms
- Box Plots
- Heatmap
- Pairplot
- Scatter Plot

---

## 🔍 Key Insights

- Female passengers had a significantly higher survival rate than males.
- First-class passengers were more likely to survive.
- Younger passengers had slightly better survival chances.
- Higher ticket fares were generally associated with higher survival.
- Most passengers boarded from Southampton.
- Passenger class showed a strong relationship with fare and survival.
